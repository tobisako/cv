## 1. 職務要約

1994年より約30年にわたり、組込み開発・モバイル・Web/SaaS・ブロックチェーン・マーケティング技術・AIまで領域を広げてきた**プレイングマネージャー型のITプロフェッショナル**です。SE→プリセールス→PM→CTOへとキャリアを伸ばし、2017年8月にフリーランス独立、**2020年7月に GridWorld 合同会社を設立**。代表社員として、上流コンサルから実装・運用・教育まで一貫して提供しています。

GridWorldは **WEB・デジタル領域の実装力と技術支援に強み** を持ち、メガバンク・大手証券・大手生活用品メーカー・エネルギー系スタートアップなど多様なクライアントを支援。**カンボジア拠点のオフショアメンバー（プノンペン／シェムリアップ／プレイベン）** および **AIテクノロジー** を駆使した新たな価値創出にも注力しています。

直近の数年は **マーケティング技術領域（GA4／サーバーサイドGTM／DataOps／BI）** の設計・移行・運用を中心に大手金融・住宅・交通・通信など多業種に提供。2025年以降は **生成AI／Claude Code の実務活用** に軸足を移し、**MCPサーバー自作3件・AIエージェント5体のマルチテナント運用・freee API会計自動化・RAG（PostgreSQL＋pgvector）構築** など「AI×実装×運用」の一貫提供を実現。**2025年10月にはカンボジアオフショア特化のAIコンサル・実装会社「GridJapan株式会社」を設立**（代表取締役）し、東証グロース上場のEdTech企業と業務委託基本契約を締結、上場企業との取引実績を確立しています。

過去には株式会社アクターリアリティーで CTO 相当として技術部門を率い、**会社売上を4億→7億へ押し上げ社長賞を受賞**。ブロックチェーン・ハッカソン「Decrypt Tokyo 2019」優勝、ブロックチェーン大学校（FLOC）での講師登壇、Qiita 約12本・OSS活動など発信実績も多数。

### 7-1. 公開リポジトリ（オリジナル：3件）

#### 🌐 [GridWorldRAG（WinServerRAG）](https://github.com/GridWorldOrganization/GridWorldRAG)

> Windows 常駐の RAG（Retrieval-Augmented Generation）サーバー。Google Drive の共有フォルダをセマンティック検索し、Claude Cowork / Claude Desktop / 任意の MCP クライアントから **MCP（Model Context Protocol）経由で遠隔検索** できます。

- **実測性能**：warm search 840ms（AWS API Gateway 経由、リランカー込み、RTX 4070 SUPER）
- **構成**：PostgreSQL 17 + pgvector、CUDA 12.4、4並列ワーカー、自動差分同期、ゾンビGC
- **セキュリティ**：Basic Auth + per-drive 分離スキーマ + PBKDF2-SHA256 + AWS IAM 最小権限
- **位置づけ**：自社運用中の業務ツールのソース公開

#### 🌐 [ClaudeWorkMulti](https://github.com/GridWorldOrganization/ClaudeWorkMulti)

> ChatWork × Claude（Anthropic API / Claude Code CLI）による **複数AI人格生成システム**。各メンバーが独自ペルソナ・会話モード・ルーム別設定を持ち、AWS SQS 経由で並列スレッド処理。

- ChatWork → API Gateway → Lambda → SQS → ポーラー（Win PC）→ Claude Code CLI → ChatWork
- 5種類の会話モード（ログ／業務／ペルソナ／ペルソナ+／反抗期）、ルーム別設定、Google Workspace連携
- AI同士の会話、フォローアップ自動検知、ホワイトリスト・安全停止対応

#### 🌐 [CambodiaJapanEarthMap](https://github.com/GridWorldOrganization/CambodiaJapanEarthMap)

> 都市名指定で地球儀風（正射影）地図画像を生成する Python ツール。日本とカンボジアに旗マーカーを描画。多言語フォント（日本語＋クメール語、harfbuzz 結合文字対応）。
